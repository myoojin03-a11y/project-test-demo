# 고객상담관리센터

고객상담관리센터는 상담원이 고객 정보와 상담 내역을 보다 쉽게 확인하고 관리할 수 있도록 만든 내부용 웹 시스템입니다.  
고객별 상담 이력을 한눈에 확인할 수 있고, 상담 내용을 등록하거나 수정하면서 현재 처리 상태까지 관리할 수 있도록 구성했습니다.

## 프로젝트 개요

상담 업무에서는 고객 정보 확인과 이전 상담 내역 조회가 빠르게 이루어져야 합니다.  
이 프로젝트는 이러한 업무 흐름을 반영하여, 상담원이 보다 편리하게 상담 기록을 관리할 수 있도록 구성한 웹 기반 내부 관리 시스템입니다.

## 사용 기술

- HTML5
- CSS3
- JavaScript

## 실행 환경

- Chrome 등 최신 웹 브라우저에서 실행할 수 있습니다.
- 별도의 프로그램 설치나 서버 설정 없이 바로 사용할 수 있습니다.

## 실행 방법

1. 프로젝트 폴더 안의 `index.html` 파일을 실행합니다.
2. 브라우저에서 고객상담관리센터 메인 화면이 정상적으로 나타나면 실행이 완료된 것입니다.

## 주요 기능

- 고객 목록 조회 및 검색
- 고객별 문의 상세 조회
- 상담 내용 등록 및 수정
- 상담 진행 상태 관리
- 접수 / 처리중 / 완료 상태 관리
- 금일 상담 현황 확인
- 고객 이름 검색을 통한 문의 고객 조회
- 자주 묻는 질문 페이지 제공

## 사용 방법

1. 고객 목록에서 원하는 고객을 선택합니다.
2. 선택한 고객의 상담 기록을 확인합니다.
3. 필요한 경우 상담 내용을 새로 등록하거나 기존 내용을 수정합니다.
4. 상담 상태를 변경하여 처리 현황을 관리합니다.

## 참고 사항

- 화면이 정상적으로 보이지 않을 경우 브라우저를 새로고침해 주세요.
